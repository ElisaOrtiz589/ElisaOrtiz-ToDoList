# Proyecto: To-Do List

Este es un proyecto de una aplicación web simple para la gestión de tareas pendientes (To-Do List), desarrollado como parte de un desafío para validar conocimientos en manipulación dinámica del DOM con JavaScript.

## Descripción

La aplicación permite a los usuarios agregar, eliminar y marcar tareas como completadas. La interfaz se actualiza dinámicamente para reflejar el estado actual de la lista de tareas, mostrando un resumen del total de tareas y cuántas de ellas han sido realizadas.

## Características Implementadas

El proyecto cumple con todos los requerimientos solicitados, sumando un total de 10 puntos:

-   **[x] Agregar Tareas (2 puntos):** Se pueden agregar nuevas tareas a través de un campo de texto y un botón. La lista se actualiza automáticamente.
-   **[x] Borrar Tareas (2 puntos):** Cada tarea tiene un botón para ser eliminada. Al hacerlo, la tarea desaparece del arreglo de datos y de la vista.
-   **[x] Conteo Total de Tareas (2 puntos):** Un contador muestra el número total de tareas y se actualiza en tiempo real al agregar o eliminar elementos.
-   **[x] Marcar Tareas como Completadas (2 puntos):** Cada tarea cuenta con un botón para cambiar su estado entre "pendiente" y "realizada". Visualmente, las tareas completadas se muestran tachadas.
-   **[x] Conteo de Tareas Realizadas (1 punto):** Un segundo contador muestra el número de tareas que han sido marcadas como completadas, actualizándose con cada cambio de estado.
-   **[x] Estado Inicial (1 punto):** La aplicación se carga inicialmente con 3 tareas de ejemplo, demostrando el recorrido y renderizado inicial del arreglo de datos.

## Tecnologías Utilizadas

-   **HTML5:** Para la estructura semántica de la página.
-   **CSS3:** Para el diseño y la maquetación, utilizando Flexbox para un layout adaptable y moderno.
-   **JavaScript (ES6+):** Para toda la lógica de la aplicación, incluyendo la manipulación del DOM, gestión de eventos y manejo del estado de las tareas en un arreglo de objetos.
-   **Font Awesome:** Para los iconos utilizados en los botones de acción.

## ¿Cómo ejecutar el proyecto?

2.  Navega a la carpeta raíz del proyecto.
